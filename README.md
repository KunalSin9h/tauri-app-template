# Tauri App Template

Features

1. Vite + React
2. Tailwind Css
3. pnpm
4. Prettier
5. Eslint
6. [Rspc](https://www.rspc.dev/) with Tauri + React Integration

8. [Husky](https://typicode.github.io/husky/) for pre commit
9. GitHub Workflow
10. Tauri vscode debug
11. [Shadcn UI](https://ui.shadcn.com/)
